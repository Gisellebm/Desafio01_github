Desafio 01 - Primeiro projeto no Github

Criar um novo projeto contendo pelo menos um arquivo. Fazer modificações no projeto salvando pelo menos três commits nele, e depois salvar esse projeto como um repositório no Github.

Passos básicos para criação de um novo projeto

Git Bash

  • git init
  • git add .
  • git commit -m "Mensagem explicativa"
  • git branch -M main

GitHub

  • Criar um novo repositório no github
  • copiar o endereço do repositório criado para usar no git bash: git@github.com:seuusuario/seurepositorio.git

Git Bash

  • Adicionar o link do repositório remoto com o do computador.
  • git remote add origin git@github.com:seuusuario/seurepositorio.git
  • git push -u origin main
